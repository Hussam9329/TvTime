# شنو تروح تجيك بعد التطبيق

## أول فحص — قاعدة البيانات

شغّل:

```bash
bun run db:verify
```

المتوقع: نجاح `integrity_check`، عدم وجود foreign-key errors، عدم وجود صفوف Media مكررة، وعدم وجود حالات أو مرايا متناقضة.

## ثاني فحص — فيلم

1. أضف فيلمًا إلى Watchlist.
2. قيّمه قبل مشاهدته.
3. تأكد أنه بقي Planned ولم يظهر في Watched/Recently.
4. علّمه كمشاهد بدون تغيير التقييم.
5. أزل المشاهدة وتأكد أنه عاد Planned وبقي تقييمه.

## ثالث فحص — مسلسل

1. تابع مسلسلًا.
2. علّم حلقة واحدة؛ الحالة Watching.
3. علّم كل الحلقات المتاحة لمسلسل مستمر؛ الحالة Up to date.
4. أزل حلقة؛ الحالة تعود Watching.
5. تأكد أن الحلقات المستقبلية غير قابلة للتعليم ولا تدخل بالحساب.

## رابع فحص — تطابق الصفحات

افتح نفس العنصر في:

- Home
- Library
- Movie/TV Details
- TV Tracking
- Recently Watched

المتوقع: لا يوجد اختلاف في الحالة أو العدادات بين الصفحات.

## خامس فحص — الثبات

أعد تشغيل السيرفر والمتصفح. المتوقّع أن تبقى الحالات كما هي لأن مصدرها SQLite/Prisma، وليس localStorage.
