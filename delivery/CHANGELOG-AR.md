# سجل التغيير

## Added

- محرك حالات TV مركزي مستقل عن التقييم.
- طبقة سيرفر لجلب Metadata والحلقات المعروضة والتحقق من عمليات الحلقة.
- إصلاح توافقي لسجلات المسلسلات المكتملة القديمة التي لا تحتوي سجلات حلقات.
- اختبارات وحدات لمحرك الحالات.
- فاحص ثابت لحماية PostgreSQL وملفات النسخة الأم وقوانين TVM-03/04/05.

## Changed

- APIs الحلقات وTV Tracking والإحصائيات والاستيراد.
- Hooks المشاهدة والتقييم والقائمة والمتابعة والتقدم.
- صفحات TV Details وTV Tracking وLibrary وHome وCalendar.
- Recently/Watched/Watchlist صار فصلها مبنياً على بيانات المشاهدة والحالة الفعلية.

## Disabled

- سكربتات صيانة قديمة تتجاوز محرك الحالات أو تعلم جميع الحلقات دون فحص العرض.

## Database

- لا تغيير schema.
- لا migration.
- لا db push.
- لا تغيير provider أو DATABASE_URL.
