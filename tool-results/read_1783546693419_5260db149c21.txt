     1→# CineTrack - TV Time Clone Project Worklog
     2→
     3→## Project Overview
     4→Building a complete TV Time-like movie & TV show tracking application using:
     5→- Next.js 16 with App Router
     6→- TypeScript + Tailwind CSS 4 + shadcn/ui
     7→- Prisma ORM (SQLite)
     8→- TMDB API (free) proxied through Next.js API routes
     9→- Zustand for state, TanStack Query for server state
    10→
    11→## Architecture Decisions
    12→- Single-page application pattern (only `/` route is user-visible)
    13→- View switching managed via Zustand store (home, discover, search, detail, calendar, library, stats)
    14→- TMDB API calls proxied through `/api/tmdb/*` routes (caching + no CORS issues)
    15→- Local "user" concept (client-generated user ID stored in localStorage, synced to DB)
    16→- Dark theme with rose/red accents (TV Time inspired)
