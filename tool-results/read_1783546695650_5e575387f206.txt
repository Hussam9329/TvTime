     1тЖТ     1тЖТ# CineTrack - TV Time Clone Project Worklog
     2тЖТ     2тЖТ
     3тЖТ     3тЖТ## Project Overview
     4тЖТ     4тЖТBuilding a complete TV Time-like movie & TV show tracking application using:
     5тЖТ     5тЖТ- Next.js 16 with App Router
     6тЖТ     6тЖТ- TypeScript + Tailwind CSS 4 + shadcn/ui
     7тЖТ     7тЖТ- Prisma ORM (SQLite)
     8тЖТ     8тЖТ- TMDB API (free) proxied through Next.js API routes
     9тЖТ     9тЖТ- Zustand for state, TanStack Query for server state
    10тЖТ    10тЖТ
    11тЖТ    11тЖТ## Architecture Decisions
    12тЖТ    12тЖТ- Single-page application pattern (only `/` route is user-visible)
    13тЖТ    13тЖТ- View switching managed via Zustand store (home, discover, search, detail, calendar, library, stats)
    14тЖТ    14тЖТ- TMDB API calls proxied through `/api/tmdb/*` routes (caching + no CORS issues)
    15тЖТ    15тЖТ- Local "user" concept (client-generated user ID stored in localStorage, synced to DB)
    16тЖТ    16тЖТ- Dark theme with rose/red accents (TV Time inspired)
